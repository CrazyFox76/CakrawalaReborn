# Cakrawala EduCentre

Landing page resmi **Cakrawala EduCentre** — Lembaga Pendidikan dan Pelatihan yang berkomitmen mencetak generasi unggul dan kompeten.

## Tech Stack

- **Framework:** Next.js 15 (App Router)
- **Styling:** Tailwind CSS v3
- **Bahasa:** TypeScript
- **Ikon:** lucide-react
- **Dark Mode:** next-themes

## Halaman

| Route | Keterangan |
|---|---|
| `/` | Beranda (Hero, About, Programs, Why Us, Tutors, Testimonials, Blog, FAQ, CTA) |
| `/program/[slug]` | Detail program (17 program) |
| `/blog` | Index artikel edukasi |
| `/blog/[slug]` | Detail artikel (5 artikel) |
| `/tentang-kami` | Profil lembaga |
| `/daftar` | Form pendaftaran |

## Menjalankan

```bash
npm install
npm run dev
```

Buka [http://localhost:3000](http://localhost:3000).

## Build

```bash
npm run build
npm start
```
